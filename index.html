QR Code Scanner

  • This project is an application that can read QR codes.
  • A QR code is a type of matrix barcode invented in 1994. Nowadays, QR codes are very popular due to their fast readability and greater storage capacity.
  • To implement this project, I tried to recreate some pre-implemented functionalities offered by the OpenCV library.
  • Decoding steps
    1. Image binarisation
    2. Image rotation
    3. Image dilation
    4. Image transformation into a matrix of pixels
    5. Mask computation
    6. Mask application over matrix
    7. Message decoding
  • Technologies
    • C++
